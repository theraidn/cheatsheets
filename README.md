# Cheatsheet Library

## Overview
A centralized collection of technical cheat sheets and command references for DevOps, Network, and Systems Engineering.

## Directory Structure
- **ansible/**: Configuration management playbooks and modules.
- **bash/**: Scripting patterns and debugging one-liners.
- **docker/**: Containerization commands and best practices.
- **git/**: Version control workflows and utilities.
- **kubernetes/**: Orchestration, Helm, and K8s resource management.
- **linux/**: System administration, kernel, and storage commands.
- **monitoring/**: Observability and logging tool references.
- **networking/**: Network troubleshooting and configuration.
- **security/**: IAM, hardening, and security protocols.
